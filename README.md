# PigLatin

Creating a pig latin translator to learn Sinatra

## Installation

Add this line to your application's Gemfile:

    gem 'pig_latin'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install pig_latin

## Usage

install gem with this in your Gemfile:

source 'https://rubygems.org'

gem 'sinatra'
